# tmux 入门


prefix + <command>

* prefix + n: 新窗口
* prefic + n/p: next/previous window
* prefix + w: list window


panel:

* prefix + %: 竖直切分
* prefix + :: 水平切分


session:

* tmux new -s {session-name}
* prefix + d: detattach 
* tmux list-sessions/ls
* tmux attach -t {session-name}

help: prefix + ?
